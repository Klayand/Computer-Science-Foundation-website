第一章练习

1._____is a memory type with capacitors that need to be refreshed periodically.（）

Correct Answer：B


2.In the ___method to synchronize the operation of the CPU with the I/O device,a large block of data can be passed from an I/O device to memory directly.()

Correct Answer：C


3.The methods used to describe the behavior of gates and circuits are:

Answers：

Boolean expressions,Logic diagrams and Truth tables


4.The steps of the Fetch-Execute Cycle:

Answers：

fetch the next instruction,decode the instruction,get data if needed,execute the instruction.


5.Judge true or false:The arithmetic/logic unit(AUL) is only capable of performing basic arithmetic operations.(A.True B.False)

Correct Answer：False


6.Judge true or false:We can divide the computer hardware devices into 4 basic categories:System unit,Input/Output devices,Secondary storage devices,Communications devices.(A.True B.False)

Correct Answer：True


7.Judge true or false:IC(Integrated circuits) are classified by the number of gates contained in them.(A.True B.False)

Correct Answer：True


8.The three steps in the running of a program on a computer are performed______in this specific order.()

Correct Answer：C


9.The____controller features a parallel interface and daisy-chained connection for I/O devices.（）

Correct Answer：A


10.____is a type of memory in which the user,not the manufacturer,stores programs that cannot be overwritten.（）

Correct Answer：B
